## 🌟 使用说明

1. **本地运行**：
   - 直接在浏览器中打开 `index.html` 文件
   - 或使用本地服务器运行

2. **导航使用**：
   - 点击右上角头像显示导航菜单
   - 左侧边栏提供快速联系方式
   - 页面平滑滚动到对应区域

3. **项目查看**：
   - 点击项目卡片可跳转到详细页面
   - 支持在新标签页中打开

4. **艺术作品**：
   - 点击生活页面的"艺术星球"按钮
   - 查看Lisa的绘画作品集
